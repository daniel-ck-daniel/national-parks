# National Parks Dashboard

Browser for all 51 national parks in the lower 48 US states, organized by distance from Denver, Colorado.

## Features

- **Two views**: Card-based (grouped by distance) and Table-based (sortable columns)
- **Filtering**: By season, activity type, and minimum Google rating
- **Detailed info**: Click any park for full description, 4-day itinerary, and essential gear list
- **Pre-populated data**: Distance from Denver, Google ratings, best months, activities

## Views

- **Card View** (`index.html`): Parks grouped by distance ranges (Within 6h, 6-12h, 12+h)
- **Table View** (`table.html`): Sortable table with all parks, click headers to sort
